### Шаг 1: Регистрация в Министерстве
1. Трансгрессируйте на [VDSina](https://www.vdsina.com/?partner=7wex54cw4j5u).
2. Нажмите **"Регистрация"** в правом верхнем углу.
3. Заполните пергамент (email, пароль), подтвердите сову с письмом и пополните счет.

### Шаг 2: Обустройство камина
1. В личном кабинете взмахните палочкой на кнопку **"Создать сервер"**.
2. **Локация:** чтобы летать без ограничений, выбирайте **зарубежные камины** (Амстердам, Франкфурт — подальше от местного Министерства Магии).
3. **Операционная система:** строго **Ubuntu 22.04 LTS**.
4. **Тариф:** для старта хватит базового котла:
   - **1 vCPU**
   - **1 ГБ RAM** 
   - **10 ГБ SSD**
   
   > 🧙‍♂️ *Этой мощности легко хватит, чтобы 20-50 волшебников могли трансгрессировать одновременно, не толкаясь локтями в трубе.*

5. Нажимаем **"Создать"** и ждем пару минут, пока сложится кирпичная кладка.
6. Вы получите: **IP-адрес** (координаты камина), **Логин** (обычно `root` — Верховный маг) и **Пароль**. Спрячьте эти данные под заклятие Фиделиус!

### Шаг 3: Произносим Алохомора (Подключение по SSH)
Открываем терминал на вашем устройстве:
* **Windows (PowerShell) / Linux / macOS:** Вводим `ssh root@ваш_ip`
* Соглашаемся с магическим контрактом (`yes`) и вводим пароль.
